# Fuel Droplet Evaporation Data Repository 

The Fuel Droplet Evaporation Data Repository is an open database dedicated to consolidating quantitative experimental data on fuel droplet evaporation under a wide range of thermodynamic and transport conditions.

Fuel droplet evaporation is a fundamental process in combustion science, multiphase flow, atomization, and heat and mass transfer. However, existing experimental data are widely scattered across the literature, often reported only in graphical form and lacking standardized formats for direct comparison or model validation.

This repository addresses this gap by systematically digitizing, organizing droplet evaporation datasets from published studies, enabling direct access to benchmark data.

## 🎯 Objectives

- To systematically collect and curate experimental droplet evaporation data from peer-reviewed literature.
- To provide a unified, machine-readable database for model validation and benchmarking.
- To enable rapid comparison between theoretical models, numerical simulations, and experimental observations.
- To support the development of next-generation droplet evaporation and multiphase transport models.

## 🔬 Scope of the Database

The repository currently includes (and will be continuously expanded to):

- Hydrocarbon fuels (e.g., n-dodecane, n-heptane)
- Oxygenated fuels (e.g., methanol, ethanol)
- Reactive and light species (e.g., ammonia)
- Wide ranges of ambient conditions:
  - Subcritical
  - Transcritical
  - Supercritical regimes
- Variable ambient pressure, temperature, and gas composition

## 📊 Data Structure

All datasets are organized in a standardized format including:

- Time evolution of droplet diameter (D, D²)
- Ambient thermodynamic conditions (T, P)
- Initial droplet size and composition
- Reference metadata and DOI links
